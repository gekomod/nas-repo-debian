NAS Debian Repository

Repozytorium aplikacji dla serwerów NAS

Instalacja

Aby dodać repozytorium do swojego systemu, wykonaj poniższe polecenia:

# Dodaj klucz GPG
wget -qO - https://repo.naspanel.site/KEY.gpg | sudo apt-key add -

# Dodaj repozytorium
echo "deb [arch=amd64] https://repo.naspanel.site/ stable main" | \
sudo tee /etc/apt/sources.list.d/nas-repo.list

# Aktualizuj i instaluj
sudo apt update
sudo apt install nas-panel nas-webserver
Kopiuj polecenia

Dostępne pakiety

  • nas-panel
    Panel zarządzania serwerem NAS przez przeglądarkę
  • nas-webserver
    Lekki serwer WWW zoptymalizowany dla systemów NAS
  • nas-storage
    Narzędzia do zarządzania przestrzenią dyskową
  • nas-backup
    Zaawansowane narzędzia do tworzenia kopii zapasowych
