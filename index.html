NAS Debian Repository

Repository for NAS applications

Installation

# Add GPG key
wget -qO - https://repo.naspanel.site/KEY.gpg | sudo apt-key add -

# Add repository
echo "deb [arch=amd64] https://repo.naspanel.site/ stable main" | \
sudo tee /etc/apt/sources.list.d/nas-repo.list

# Update and install
sudo apt update
sudo apt install nas-panel nas-webserver

Available Packages

  • nas-panel - Web management panel
  • nas-webserver - Lightweight web server
